# Port Doctor 🩺

**Universal ROM Port Bug Detector & Auto-Fixer**

Works on ANY rooted Android device with a ported ROM.

## Features

- 🔍 Auto-detects 17+ common port issues
- ⚡ Generates custom fix scripts
- 📦 Builds flashable Magisk modules
- 📊 Creates XDA-ready detailed reports
- 🌐 Community database for vendor files
- 📤 Share fixes with the community

## Supported Devices

**ALL rooted Android devices** — Realme, Xiaomi, Samsung, OnePlus, Motorola, Poco, Redmi, Nothing, and more.

## How to Use

1. Download the APK from [Releases](https://github.com/YOUR_USERNAME/Port-Fixer/releases)
2. Install and grant root access
3. Set your device and donor device
4. Tap "Scan Device for Issues"
5. Select which issues to fix
6. Tap "Generate Fixes" then "Build Module"
7. Flash the generated zip in Magisk
